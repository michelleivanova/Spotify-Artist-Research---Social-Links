# Rappers Social Links - Summary

## File Created
**Location**: `Final_Social Links/rappers_final.xlsx`  
**Size**: 42 MB  
**Format**: Multi-sheet Excel file (matching female_singers_final.xlsx format)

## File Structure

### Sheet 1: "All Data"
- **Rows**: 50,000 artists
- **Columns**: 204 columns
- **Content**: All original Soundcharts data PLUS social media columns at the end

### Sheet 2: "Social Links"
- **Rows**: 50,000 artists
- **Columns**: 14 columns
- **Content**: Artist name, country, and all social media links
- **Columns**: Artist, Artist country, instagram_url, instagram_handle, tiktok_url, tiktok_handle, youtube_url, youtube_channel_id, soundcloud_url, soundcloud_handle, twitter_url, twitter_handle, facebook_url, website_url

## Social Media Coverage

Successfully generated social links for **49,877 out of 50,000 artists (99.8%)**:

- **Instagram**: 49,877 URLs (99.8%)
- **TikTok**: 49,877 URLs (99.8%)
- **YouTube**: 49,877 URLs (99.8%)
- **Twitter/X**: 49,877 URLs (99.8%)
- **SoundCloud**: 49,877 URLs (99.8%)
- **Facebook**: 49,877 URLs (99.8%)

**Total fields populated**: 548,647

## Notes

- All social links are **auto-generated** based on artist names
- The `lookup_status` field is set to "auto_generated" for all entries
- The `Artist_Type` field is set to "Rapper" for all entries
- Common rapper prefixes (Lil, Young, Big, The) are removed when creating handles
- 123 artists (0.2%) did not have social links generated (likely due to empty/invalid names)
